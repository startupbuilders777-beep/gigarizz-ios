## APP STORE CONNECT SETUP CHECKLIST

### Per Locale (repeat for each market)
```
□ 1. Login to App Store Connect → App → Localizations
□ 2. Add locale: select from list (de-DE, fr-FR, es-ES, it-IT, ja-JP, pt-BR)
□ 3. Add localized screenshots (5 iPhone frames per locale)
□ 4. Update subtitle (localized version)
□ 5. Update short description (localized version)
□ 6. Update full description (localized version)
□ 7. Update keyword field (localized terms)
□ 8. Update promo text (optional, can update without review)
□ 9. Set primary locale if not already: English (U.S.) — en-US
□ 10. Submit for review (only needed if metadata changes trigger it — promo text does not)
```

### Screenshot Localization Notes
- **UK/Australia/Canada:** Use same US English screenshots (no text changes needed)
- **Germany:** Add German captions to bottom frame (or use same visuals + en-US text is fine)
- **France/Spain/Italy:** Same as Germany — screenshots can stay English if copy is localized
- **Japan:** Localized screenshots strongly recommended — Japanese users respond better
- **Brazil:** Portuguese screenshots recommended

### Priority for Screenshot Localization
```
LOW   — UK, Australia, Canada (English, same screenshots as US)
MED   — Germany, France, Spain, Italy (can use en-US screenshots with localized text)
HIGH  — Japan, Brazil (localized screenshots significantly improve conversion)
```
