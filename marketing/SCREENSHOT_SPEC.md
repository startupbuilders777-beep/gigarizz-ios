## PRODUCTION NOTES

### Photo shoot / generation requirements
1. **Same model across ALL styles** вҖ” pick 1 person, shoot them 10Г— with consistent framing
2. **Natural, genuine expressions** вҖ” not model-perfect, relatable
3. **Consistent outfit** вҖ” white/gray tee for neutral base
4. **Varied backgrounds** вҖ” cafe, golden hour outdoors, urban wall, studio backdrop, rooftop, sports field
5. **Rizz Scores** вҖ” use consistent scale: 4.2-9.1 across before/after

### Frame compositing workflow
1. Generate/capture base photos
2. Add Rizz Score badges (Figma template with auto-layout)
3. Add typography overlays
4. Apply brand colors via Figma styles
5. Crop to exact dimensions
6. Export PNG (no alpha) at 2x resolution for Retina

- [ ] iPhone 16 Pro Max: 2556Г—1179 (5 frames)
